# Protected File Rules

## Protected paths
The following paths are protected and cannot be modified during standard execution:
- `project/vision/*`
- `project/docs/features/*`
- `project/docs/task_groups/*`
- `project/docs/priorities/*`

## Restricted paths
The following paths are restricted and may be modified only by authorized planning updates:
- `project/docs/tasks/*`

## Allowed paths
The following paths are allowed for execution outputs:
- `project/app/*`
- `project/evidence/*`

Violation = invalid execution.
