# Start Here

1. Read `README.md`.
2. Confirm this repository controls one project under `project/`.
3. Read `docs/control-model.md` and `docs/routing.md`.
4. Read framework rules in `framework/rules/` before making changes.
5. If execution is requested, consume `project/now/description.md`, `project/now/prompt.md`, and `project/now/metadata.json`.
6. Validate that protected boundaries were respected before marking work complete.

This framework governs repository truth and execution alignment. It does not govern AI internals.
